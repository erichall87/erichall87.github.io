Dynamic Mirror Descent(DMD) - Demo code for various experiments using DMD and variants to reproduce experiments from our IEEE-JSTSP Paper, including Dynamic Textures, Compressed Sensing with moving support and DMD with additive dynamics in exponential families.
Hawkes Process - Demo code for experiments to learn network and rates from streaming data related to the Hawkes process as described in our IEEE-IT paper.
