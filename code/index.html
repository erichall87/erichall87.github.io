--- layout: layout title: Academic Code ---
  Dynamic Mirror Descent(DMD) - Demo code for various experiments using DMD and variants to reproduce experiments from our [JSTSP Paper](http://ieeexplore.ieee.org/xpls/abs_all.jsp?arnumber=7044563), including Dynamic Textures, Compressed Sensing with moving support and DMD with additive dynamics in exponential families.
  Hawkes Process - Demo code for experiments to learn network and rates from streaming data related to the Hawkes process as described in our [IEEE-IT paper](http://ieeexplore.ieee.org/xpls/abs_all.jsp?arnumber=7469837).
